Arrays

Imagine this: you're a teacher, tasked with storing the marks of all 25 students in your class. Each student has a number, from 1-25, and a decimal mark, from 0-100. How would you store this information?

Well, one way to do this would be to create a variable for each student. This would allow you to access each student individually, and would store their marks correctly. But what if you had fifty, or even a hundred students? That would be a LOT of variables, and would take far too much work. So what could you do instead of this?

A much more effective way of dealing with this information would be to use an array. An array is a data structure that stores data in the form of a list. Arrays have a fixed size, meaning once you create a list, you cannot add or remove items from it. Data stored in an array is accessed by its index, which starts at 0. Arrays can be created for any data type, including instances of other classes. Arrays are also useful for manipulating data. You can sort, search, and update the data stored in an array, making it quite useful in a variety of scenarios.

Arrays VS ArrayLists

Both arrays and arrayLists store a bunch of data under a single variable name, and allow individual objects in the array/arrayList to be accessed using an index. An array can be created for any data type, and so can arrayLists. So what's the difference? The table below highlights some of the main differences between these two data types.

Syntax

Now that you know what an array is and why you would use one, what's the syntax? Just like all other data types, you start by declaring an array. You do this by first typing the data type, then adding square brackets afterwards to indicate that it is an array. Then, you initialize it and provide the size of the array. An example of this process is in the photo to the left.

For creating an ArrayList, the syntax is slightly different. It is mandatory to import java.util.ArrayList before creating an ArrayList. Once you do, you can declare a new arraylist by following this syntax:
ArrayList arrayListName = new ArrayList();
If you wish, you could provide a size within the brackets, but it is not necessary. If a size is not given, the default capacity is ten.

Common Errors

There are a few things that you should keep in mind when using arrays. Firstly, arrays are zero-indexed - the first object in an array is always at an index of zero, not one. Because of this, the last object in an array is at an index of array.length-1, not at an index of array.length.

Additionally, when looping through arrays, it is important to ensure you set the correct loop end case correctly, otherwise you may either go out of bounds or skip elements. For example, if an array has a size of length, trying to access the data at index == length will cause an IndexOutOfBounds exception.

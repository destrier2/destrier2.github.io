Object Oriented Programming

What is Object Oriented Programming?

Object Oriented Programming is a model that allows code to represent real-life objects. OOP breaks down the world into different levels, which are compared in the following chart.

Classes are the broadest, most abstract level of OOP. They are unspecific templates that are used to create objects.

An instance is a specific object that is created from a class. In this case, "Maple" is an instance of the class "Tree".

Why should I care?

Many widely-used coding languages follow OOP, including Java, C++, and Javascript. Understanding OOP is essential to understanding these languages, since the OOP format is the backbone of these languages.

Inhertiance

Now that you understand what classes and objects are, let's take a look at inheritance. Inheritance is also an important concept in OOP, and is also designed to echo real life. In OOP inheritance, it is when you create subclasses (children) of a superclass (parent). In real life, when a parent gives birth to a child, the child also inherits some of the attributes of the parent - they may share eye color, habits, or physical traits. When it comes to code, a subclass will inherit any public methods or fields. A subclass can also override any of these methods with its own version.
